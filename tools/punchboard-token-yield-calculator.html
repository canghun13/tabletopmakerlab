Die-cut nesting, safety gaps, and supplier constraints can change the final production layout.

Use a grid as an early constraint

This tool divides usable board space into rows and columns after margin and gutter. It is most useful when deciding whether a token set needs another board.

Method

Available width and height are divided by token footprint plus gutter. Circle mode uses the diameter as both dimensions for a conservative grid.

Example

Try your target token count with one board, then add a board rather than reducing spacing below a sensible production assumption.

Limits

Real die-cut layouts can nest circles and irregular shapes more efficiently. Confirm with the manufacturer.
