How the calculation works

Copies required to recoup equals the advance divided by royalty per copy, rounded up. Royalty generated before that point offsets the advance; only the excess is shown as payable.

Input guide and example

An advance is commonly a payment against future royalties, rather than an extra bonus. A $5,000 advance with $1.00 royalty per copy needs 5,000 copies to recoup. Agreements can define recoupment differently, so read the actual wording.
